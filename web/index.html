LBM colour cycling animation viewer

P - Toggle show & hide palette
S - Toggle show & hide update spans
M - Next palette cycle interpolation method, they are as follows:
  1. Step - Disable interpolation and rotate palette in discrete steps, like Deluxe Paint.
  2. sRGB - Na√Øve linear interpolation directly in sRGB colourspace.
  3. Linear - Linear interpolation in linear space.
  4. HSLuv - Linearly interpolate in the HSLuv colour space, somewhat unstable.
  5. CIELAB - Linearly interpolate in the CIELAB (D65 white point) colour space.
[ and ] - Decrease/increase the cycling speed.

Source code for the viewer can be viewed on GitHub,
the viewer was cross-compiled for the web using Emscripten.
